# 해외직구멀티샵 - 수동 테스트 체크리스트

이 문서는 MVP 배포 전 주요 기능을 수동으로 테스트하기 위한 체크리스트입니다.

---

## 1. 홈페이지 (`/`)

### 기본 렌더링
- [ ] 히어로 섹션이 정상적으로 표시되는가
- [ ] "상품 둘러보기" 버튼이 `/products`로 이동하는가
- [ ] Features 섹션 (AI 리뷰 요약, 다국어 번역, 한국어 리뷰) 표시

### 카테고리 섹션
- [ ] 카테고리 목록이 표시되는가 (데이터 있을 때)
- [ ] 카테고리 카드 클릭 시 해당 카테고리 상품 목록으로 이동
- [ ] "전체 보기" 링크 동작 확인

### 추천 상품 섹션
- [ ] 추천 상품 (is_featured=true) 목록 표시
- [ ] 상품 카드 9:16 비율 렌더링
- [ ] 상품 카드 호버 시 영상 재생 (video_url 있는 경우)
- [ ] "더 보기" 링크 동작 확인

---

## 2. 상품 목록 페이지 (`/products`)

### 상품 그리드
- [ ] 반응형 그리드 동작 (데스크톱 4열, 태블릿 3열, 모바일 2열)
- [ ] 상품 카드 정상 렌더링 (썸네일, 제목, 가격, 평점)
- [ ] 호버 시 영상 재생

### 필터링
- [ ] 카테고리 필터 동작
- [ ] 플랫폼 필터 동작 (아마존, 알리, iHerb 등)
- [ ] 가격 범위 필터 동작
- [ ] 평점 필터 동작

### 정렬
- [ ] 최신순 정렬
- [ ] 가격 낮은순/높은순 정렬
- [ ] 평점순 정렬
- [ ] 조회수순 정렬

### 검색
- [ ] 키워드 검색 동작
- [ ] 검색 결과 표시
- [ ] 검색어 없을 때 전체 상품 표시

### 페이지네이션
- [ ] 페이지 이동 동작
- [ ] 현재 페이지 표시
- [ ] 총 상품 수 표시

## 3. 상품 상세 페이지 (`/products/[slug]`)

### 상품 정보
- [ ] 상품 이미지/영상 표시
- [ ] 상품명, 가격, 평점 표시
- [ ] 플랫폼 정보 표시
- [ ] 구매 링크 버튼 동작 (외부 사이트로 이동)

### AI 요약 박스
- [ ] AI 요약 표시 (캐시된 경우)
- [ ] "AI 요약 생성" 버튼 동작
- [ ] 긍정/부정 포인트 목록 표시
- [ ] 추천 대상 표시

### 외부 리뷰 목록
- [ ] 외부 리뷰 목록 표시
- [ ] 원문/번역 토글 동작
- [ ] 리뷰어 정보 표시 (이름, 국가, 날짜)
- [ ] 평점 표시

### 사용자 리뷰
- [ ] 사용자 리뷰 목록 표시
- [ ] 리뷰 작성 폼 표시 (로그인 시)
- [ ] "도움됨" 투표 동작
- [ ] 내 리뷰 수정/삭제 동작

### 관련 상품
