# 해외직구멀티샵 - Product Requirements Document (PRD)

## 1. 제품 개요

### 1.1 비전
**"전세계 인기 상품의 리뷰를 AI가 한눈에 요약해드립니다."**

해외직구멀티샵은 여러 해외 쇼핑몰에 흩어진 상품 정보와 리뷰를 한 곳에 모아, AI 기술로 요약하여 제공하는 큐레이션 플랫폼입니다.

### 1.2 목표 사용자
- **Primary**: 2030 직장인 해외직구 이용자
- **특징**: 시간이 부족하고, 효율적인 쇼핑을 원함

### 1.3 핵심 가치
1. **시간 절약**: 30분 → 5분으로 구매 결정 시간 단축
2. **언어 장벽 제거**: AI 자동 번역 및 요약
3. **신뢰성 강화**: 자체 한국어 리뷰 시스템

---

## 2. 핵심 기능 (MVP Scope)

### 2.1 상품 검색 및 필터링
- 카테고리별 상품 탐색
- 가격대, 평점 필터
- 키워드 검색

### 2.2 AI 리뷰 요약 시스템
- 수백 개 리뷰 자동 요약
- 긍정/부정 포인트 분리
- 추천 대상 명시

### 2.3 다국어 번역
- 영어, 중국어, 일본어 → 한국어
- 원문/번역 토글

### 2.4 자체 리뷰 시스템
- 한국 사용자 리뷰 작성
- 별점 및 사진 첨부
- "도움됨" 투표

### 2.5 위시리스트
- 상품 저장
- 가격 변동 알림 (Phase 2)

### 2.6 관리자 기능
- 상품/카테고리 관리
- 외부 리뷰 수동 입력
- AI 요약 트리거

## 4. 사용자 스토리

### US-001: 상품 검색
> "2030 직장인으로서, 나는 여러 사이트를 방문하지 않고 한 곳에서 상품을 검색하고 싶다."

### US-002: AI 리뷰 요약
> "바쁜 직장인으로서, 나는 100개가 넘는 리뷰를 읽지 않고 핵심만 파악하고 싶다."

### US-003: 리뷰 작성
> "해외직구 경험자로서, 나는 다른 한국 사용자들에게 실제 후기를 공유하고 싶다."

---

## 5. 데이터 모델

### 핵심 테이블
```
users           - Clerk 연동 사용자
categories      - 상품 카테고리
products        - 상품 정보 (썸네일, 영상, 가격 등)
external_reviews - 해외 리뷰 (수동 입력)
user_reviews    - 자체 한국어 리뷰
ai_summaries    - AI 요약 결과 캐싱
wishlists       - 위시리스트
```

## 6. UI/UX 특징

### 6.1 릴스 스타일 상품 목록
- **비율**: 9:16 (세로형 썸네일)
- **호버**: 영상 자동 재생 (무음)
- **클릭**: 전체 화면 영상 재생
- **가격 정보**: 할인율(빨간색), 할인가, 원가(취소선)
- **구매자 수**: "N명구매중" 표시
- **반응형**: 
  - 데스크탑: 4열
  - 태블릿: 3열
  - 모바일: 2열

### 6.2 홈페이지 구조
```
┌─────────────────────────────────────┐
│  HeroHeader (배경 + 로고 + 검색창)   │
├─────────────────────────────────────┤
│  CategoryNavbar (카테고리 메뉴)      │
├─────────────────────────────────────┤
│  오늘의 신상품 Today's NEW          │
│  [상품1] [상품2] [상품3] [상품4]    │
│         [+ 모두보기]                │
├─────────────────────────────────────┤
│  Features (AI요약, 번역, 리뷰)       │
├─────────────────────────────────────┤
│  카테고리 / 추천 상품 / CTA         │
└─────────────────────────────────────┘
```

### 6.3 AI 요약 박스
```
┌─────────────────────────────────────┐
│  🤖 AI 리뷰 요약                    │
├─────────────────────────────────────┤
│  📊 전체 평가: ★★★★☆ 4.3/5        │
│                                     │
│  ✅ 긍정 포인트                     │
│  • 착용감이 매우 편안함             │
│  • 디자인이 세련됨                  │
│                                     │
│  ⚠️ 부정 포인트                    │
│  • 배송이 2-3주 소요               │
│  • 가격대가 높음                    │

```

## 7. MVP 성공 지표

| 지표 | 목표 |
|------|------|
| 등록 상품 수 | 50개 이상 |
| 베타 테스터 | 30명 이상 |
| AI 요약 생성 | 100회 이상 |
| 자체 리뷰 | 20개 이상 |
| 평균 세션 시간 | 3분 이상 |

---

## 8. 개발 일정

| Phase | 기간 | 주요 작업 |
|-------|------|----------|
| Phase 1 | 1주차 | 프로젝트 설정, DB 스키마, 디렉토리 구조 |
| Phase 2 | 2주차 | 상품 목록/상세, AI 요약 인터페이스 |
| Phase 3 | 3주차 | 리뷰 시스템, 마이페이지, 번역 |
| Phase 4 | 4주차 | 관리자 페이지, UI 개선, 테스트 |

---

## 9. 향후 로드맵

- 자동 크롤링 시스템
- 상품 500개 확대
- 직접 결제 연동

### Phase 3 (3-6개월)
- 모바일 앱 출시
- AI 개인화 추천
- 라이브 커머스 연동
